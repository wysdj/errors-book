## 阶段二：系统设计（1天）

### 2.1 技术选型

```
"用最少的工具做最多的事"

前端：HTML + CSS + 原生 JavaScript（不引入 React/Vue，减少学习成本）
后端：Python Flask（轻量、适合小项目）
数据库：SQLite（单文件、不需要安装数据库软件）
AI：DeepSeek Chat API（便宜、国内可用）
视觉：通义千问 VL（看图识字）
部署：阿里云 ECS（CentOS）+ systemd

为什么不选别的？
  不用 MySQL → SQLite 就是硬盘上一个文件，拷走就是备份
  不用 React  → 这个项目 3 个页面，原生 JS 够了
  不用 Docker → 一台服务器只跑一个服务，没必要容器化
```

### 2.3 数据库设计

```sql
-- 一个表管所有错题（简单够用）
CREATE TABLE errors (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    question_text   TEXT,           -- 题目
    subject         TEXT,           -- 学科
    chapter         TEXT,           -- 章节
    topics          TEXT,           -- 知识点（逗号分隔）
    error_reason    TEXT,           -- 错误原因
    reference_answer TEXT,          -- 参考答案
    image_path      TEXT,           -- 图片路径
    mastered        INTEGER DEFAULT 0,  -- 0未掌握/1已掌握
    created_at      TEXT
);

-- 复习记录表
CREATE TABLE reviews (
    id        INTEGER PRIMARY KEY,
    error_id  INTEGER,              -- 关联错题
    passed    INTEGER DEFAULT 0,
    created_at TEXT
);
```

### 2.4 API 设计

```
POST   /ocr              拍照识别（上传图片 → 返回文字+分类）
GET    /errors           错题列表（支持筛选、分页）
POST   /errors           新增错题
GET    /errors/<id>      查看单条
DELETE /errors/<id>      删除
POST   /errors/<id>/master  切换掌握状态

POST   /review/start     开始费曼复习
POST   /review/answer    提交复习回答
POST   /review/hint      获取提示

GET    /stats            统计数据
GET    /backup           下载备份
GET    /export/csv       导出CSV
```

**关键设计决策**：所有 API 返回 JSON 格式，前端用 `fetch()` 调用。

### 3.3 关键技术难点解决

**难点 1：PaddleOCR 在 Windows 上装不上**
```
原因：PaddlePaddle 3.x 有 oneDNN bug
解决：降级到 Python 3.9 + PaddleOCR 2.7.3 + PaddlePaddle 2.6.2
```

**难点 2：OCR 太慢（15秒）**
```
原因：OCR → VL → AI清洗，三步串行太慢
解决：直接用 VL 模型一步识别，砍掉 OCR 和清洗，变 3 秒
```

**难点 3：数学公式不显示**
```
原因：KaTeX 需要字体文件，只下了 CSS/JS 没下 fonts/
解决：下载 20 个 .woff2 字体文件到服务器
```

**难点 4：PWA 缓存旧版本**
```
原因：Service Worker 缓存了旧 JS，代码更新不生效
解决：sw.js 改为 skipWaiting + clients.claim，不缓存
```

## 阶段五：部署上线（半天）

### 5.1 部署步骤

```bash
# 1. 服务器装 Python 环境
yum install -y python39 mesa-libGL
python3.9 -m venv /opt/errorbook/venv

# 2. 安装依赖
/opt/errorbook/venv/bin/pip install flask paddleocr pillow

# 3. 上传代码
scp -i key.pem * root@121.40.212.86:/opt/errorbook/

# 4. 配 systemd 服务（开机自启 + 崩溃重启）
cat > /etc/systemd/system/errorbook.service << EOF
[Unit]
Description=408 错题管理器
After=network.target
[Service]
WorkingDirectory=/opt/errorbook
Environment=DEEPSEEK_KEY=sk-...
ExecStart=/opt/errorbook/venv/bin/python3.9 ocr_server.py
Restart=always
[Install]
WantedBy=multi-user.target
EOF
systemctl enable --now errorbook

# 5. 阿里云安全组开端口 8901

# 6. 定时备份
echo "0 3 * * * cp /opt/errorbook/errors.db /opt/errorbook/backups/errors_\$(date +\%Y\%m\%d).db" | crontab -
```
